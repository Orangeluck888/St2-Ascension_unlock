# 杀戮尖塔2 - 多人游戏进阶难度解锁模组

## 简介

这个模组可以让你直接选择任何进阶难度（1-10级），无需逐级解锁。特别适合多人联机时，队友之间进阶等级不同的情况。

## 安装方法

1. 下载 `ascension_unlock.dll` 和 `ascension_unlock.pck` 两个文件
2. 找到游戏安装目录（Steam右键游戏 → 管理 → 浏览本地文件）
3. 在游戏目录下创建 `mods` 文件夹（如果没有的话）
4. 在 `mods` 文件夹中创建一个新文件夹，比如叫 `解锁所有进阶难度`
5. 把两个文件放进这个文件夹
6. 启动游戏即可
